🎮 Loja de Jogos PRO

📌 Descrição

A Loja de Jogos PRO é um sistema simples desenvolvido em Python utilizando a biblioteca Tkinter para criar uma interface gráfica. O programa permite registrar vendas de jogos, adicionar produtos ao carrinho, calcular o valor total da compra e salvar o histórico de vendas em um arquivo JSON.

🚀 Funcionalidades

- Cadastro do nome do cliente.
- Seleção de jogos disponíveis.
- Adição de jogos ao carrinho.
- Remoção de itens do carrinho.
- Cálculo automático do valor total.
- Finalização da compra.
- Salvamento automático das vendas em arquivo JSON.
- Visualização do histórico de vendas.

🛠️ Tecnologias Utilizadas

- Python 3
- Tkinter
- JSON

📂 Estrutura do Projeto

- "main.py" → Código principal do sistema.
- "vendas_jogos.json" → Arquivo onde as vendas são armazenadas.

▶️ Como Executar

1. Instale o Python 3.
2. Baixe ou copie o código para um arquivo ".py".
3. Execute o comando:

python main.py

4. A janela da loja será aberta automaticamente.

💾 Armazenamento de Dados

Todas as vendas realizadas são salvas automaticamente no arquivo "vendas_jogos.json", permitindo manter um histórico das compras efetuadas.
